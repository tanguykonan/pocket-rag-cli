<h1>Pocket-RAG</h1>

<p>
A fast, local and secure Retrieval-Augmented Generation (RAG) CLI application.
</p>

<p>
Built as a lightweight command-line tool for semantic document search and generation workflows.
</p>

</div>

<hr>

<h2>Overview</h2>

<p>
Pocket-RAG is a command-line application designed to perform Retrieval-Augmented Generation (RAG) entirely locally.  
It enables fast and secure semantic document search using <strong>ChromaDB</strong> for vector indexing and a language model API for inference.
</p>
